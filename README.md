# Dynamic Form Builder

This is a full-stack web application built for the **ReadyNest Week 1 Task**. It allows users to create, customize, and share dynamic forms, and collects responses in a real-time dashboard.

## 🚀 Features
* **Dynamic Form Builder:** A clean UI to add and customize fields (Text, Email, Dropdown, Number, etc.).
* **Live Response Dashboard:** View a real-time table of collected form submissions.
* **Client-Side Storage:** Utilizes LocalStorage for fast, database-free prototyping.
* **Modern UI:** Styled with Tailwind CSS and a sleek dark mode.

## 💻 Tech Stack
* Next.js 13
* React
* TypeScript
* Tailwind CSS

## 🛠️ How to Run Locally
1. Clone the repository: `git clone https://github.com/your-username/ready-nest-form-builder.git`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
